# Condicionales 
## Reglas basica : el `if`

Bloque condicional del if
    Esto le pertenece al bloque 
    esto tambien le pertenece 

### Sintaxis

```python
if condicion:
    accion
```

### Ejemplo 

```python
dinero = 1000
precio = 750

if dinero >= precio:
    print("venta aprovada")
```

## Condicional `else`
* No posee condicion 
* Se ejecuta si la condicion del `if` es falsa

### Sintaxis

```python
if condicion:
    accion
else:
    accion
```

```python
dinero = float(input("Ingrese dinero del cliente: "))
precio = 1200

if dinero >= precio:
    print("Puede comprar el producto")
else:
    print("No puede comprar el producto")
```

## Condicional `elif`
* Posee condicion 
* Se ejecuta si la condicion del `if` es falsa

## Condicional: `match`
* Posee condicion 
* Se ejecuta si la condicion del `if` es falsa

```python
match variable:
    case "valor1":
        accion
    case "valor2":
        accion
    case _:
        accion
```
