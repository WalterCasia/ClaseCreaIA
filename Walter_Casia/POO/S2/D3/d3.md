# Estilo profecional de python (`@property`)

`cliente.pin = 1234`
property se le llama usar decoradores para crear atributos

# syntaxis de decoradores

* **Para getter**
```python
@property
def nombre(self):
    return self._nombre

```
* **Para setter**
```python
@nombre.setter
def nombre(self, value):
    self._nombre = value
```
