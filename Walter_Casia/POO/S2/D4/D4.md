
# Manejo de Archivos en Python

## Apertura de Archivos

Para trabajar con archivos, usa la función `open()`:

```python
archivo = open('nombre.txt', 'r')  # 'r' = lectura
```

**Modos comunes:**
- `'r'` - Lectura (por defecto)
- `'w'` - Escritura (sobrescribe)
- `'a'` - Añadir contenido
- `'b'` - Modo binario (ej: `'rb'`)

## Lectura de Archivos

```python
# Leer todo el contenido
contenido = archivo.read()

# Leer línea por línea
linea = archivo.readline()

# Leer todas las líneas como lista
lineas = archivo.readlines()
```

## Escritura de Archivos

```python
archivo.write("Texto a escribir\n")
archivo.writelines(["linea1\n", "linea2\n"])
```

## Cierre de Archivos

```python
archivo.close()
```

## Mejor Práctica: Context Manager

```python
with open('archivo.txt', 'r') as archivo:
    contenido = archivo.read()
# El archivo se cierra automáticamente
```
