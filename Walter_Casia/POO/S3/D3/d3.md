## Creamos el molde fantasma(el modulo `abc`)
- `abc` = Abstract Base Clasess
- debemos marcar al menos uno de sus metodos con el decorador `@abstractmetod`
- un metodo abstracto es un metodo vacio. No tiene codigo por dentro. usamos la palabra pass

```python
#Importacion obligatoria
from abc import ABC, abstractmethod

class Documento(ABC)
    def __init_(self, titulo):
        self.titulo = titulo

    @abstractmethod
    def exportar(self):
        pass

    def mostrar_titulo(self): 
        print (f"Documento: {self.titulo} ")

class PDF(Documento):
    def exportar(self):
        print(f"Guardando ")
