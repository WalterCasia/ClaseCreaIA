# Sobrecarga de operadores

- para ensenar el `+` se usa `__add__(self, other)`
- para ensenar el `-` se usa `__sub__(self, other)`
- para ensenar el `*` se usa `__mul__(self, other)`
- para ensenar el `/` se usa `__truediv__(self, other)`
- para ensenar el `//` se usa `__floordiv__(self, other)`
- para ensenar el `%` se usa `__mod__(self, other)`
- para ensenar el `**` se usa `__pow__(self, other)`
- para ensenar el `==` se usa `__eq__(self, other)`
- para ensenar el `!=` se usa `__ne__(self, other)`
- para ensenar el `<` se usa `__lt__(self, other)`
- para ensenar el `<=` se usa `__le__(self, other)`
- para ensenar el `>` se usa `__gt__(self, other)`
- para ensenar el `>=` se usa `__ge__(self, other)`

# Importar Modulos

- para importar un modulo se usa `import modulo`
- para importar una clase de un modulo se usa `from modulo import clase`
- para importar una funcion de un modulo se usa `from modulo import funcion`
- para importar una variable de un modulo se usa `from modulo import variable`

```python
import math

print(math.sqrt(16))
```

```python
from math import sqrt

print(sqrt(16))
```

```python
from math import sqrt as raiz

print(raiz(16))
```
