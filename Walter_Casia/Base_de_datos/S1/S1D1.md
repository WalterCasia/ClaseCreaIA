# Base de datos

SQL = Structure Query Languaje o Lenguaje de consultas estructurales

## Conceptos fundamentales
- entidad: representa un objeto del mundo real que tiene relevancia en el sistema ('cliente')
- Tabla: forma en que se almacena la informacion de una entidad en la base de datos 
- filas (registros): corresponde a una instancia concreta de la entidad ('cliente en especifico')
- columna (atributos): define una caracteristica de la entidad.
(`nombre, direccion, precio`)
- llave (clave) primaria: identificador **unico** para cada fila en una tabla
- llave (clave) foranea (FK): Relacion entre tablas, enlazando con la llave primaria de la otra tabla
