# Funciones o metodos

## syntaxis

para crear una funcion se utiliza la palabra reservada def, le agregamos nombre y parentesis

```python
def nombre_funcion(parametros):
    # codigo de la funcion
```


# Parametros de una funcion

los parametros son variables que se le pasan a una funcion para que pueda realizar alguna operacion con ellos

```python
def nombre_funcion(parametro1, parametro2, parametro3):
    # codigo de la funcion
```

#return

la palabra reservada return se utiliza para retornar un valor de una funcion

```python
def nombre_funcion(parametro1, parametro2, parametro3):
    # codigo de la funcion
    return parametro1 + parametro2 + parametro3
```
