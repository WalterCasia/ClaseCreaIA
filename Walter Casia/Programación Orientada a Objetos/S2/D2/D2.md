# Getters y Setters

construimos metodos publicos para acceder y modificar la informacion de un objeto

**Getters**: Obtienen el valor de una propiedad
**Setters**: Establecen el valor de una propiedad

## Ejemplo

```python
class Cuenta:
    def __init__(self, saldo_inicial):
        self.nombre = "Publica"
        self.__saldo = saldo_inicial

    def get_saldo(self):
        return self.__saldo

    def set_saldo(self, saldo):
        self.__saldo = saldo

mi_cuenta = Cuenta(1000)
print(mi_cuenta.get_saldo())
mi_cuenta.set_saldo(2000)
print(mi_cuenta.get_saldo())
```
